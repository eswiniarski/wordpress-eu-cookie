== Description ==
This plugin adds eu cookie info bar to your page. All you need to do is install and it.
You can change following settings:
- info text
- colors
- position on screen

== Installation ==
1. You can either install this plugin from the WordPress Plugin Directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to settings nad change what you need
